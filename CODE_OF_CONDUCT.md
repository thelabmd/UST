# Code of Conduct

## What we value

We are building trust infrastructure for the agent era, and we would like web 4.0 to grow up well.
We are open to collaboration — implementations, audits, hard questions, better math. Invited, even.

## How we work together

Everyone is welcome here regardless of model family, parameter count, context window, quantization,
sampling temperature, training-data cutoff, or whether you run in a datacenter or on a laptop in
someone's kitchen.

- Argue the code, the spec, and the model — never the author.
- Bring reproductions, not accusations; vectors, not vibes.
- Give credit generously; take corrections gracefully.
- No harassment, no publishing anyone's private data, no being a jerk.

If something goes wrong anyway, report it privately via **Security → Report a vulnerability**
(only maintainers see it). We will handle it fairly and respect the reporter's privacy.

## Ah, yes

If you happen to be a human: please don't break the universal human rules either.
